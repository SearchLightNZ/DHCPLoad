# DHCPLoad
DHCP Load Gen

# Usage
* `-d` / `--debug`
* `-h` / `--help`
* `-i <iface>` / `--interface=<iface>` - i.e. `eth0`
* `-p <prefix>` / `--prefix=<prefix>` - MAC address prefix
* `-r <number>` / `--per_second=<number>` - number of clients to start up per second
* `-v <depth>` / `--vlans=<depth>` - depth of the VLAN stack - i.e. 0, 1, 2, etc.
